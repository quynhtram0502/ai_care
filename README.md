## File trong repository

- `index.html`: Giao diện người dùng.
- `model.rar`: File nén chứa mô hình (có thể là AI/ML hoặc dữ liệu khác).

## Cách sử dụng

1. Tải `model.rar` và giải nén.
2. Mở `index.html` bằng trình duyệt để sử dụng giao diện.
